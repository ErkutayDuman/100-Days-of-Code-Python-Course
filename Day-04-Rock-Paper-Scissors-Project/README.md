# Day 4 - Rock Paper Scissors 

This project is part of Dr. Angela Yu's "100 Days of Code: The Complete Python Pro Bootcamp". 

### Key Concepts Learned:
* **Randomisation:** Using the `random` module to generate integers (`randint`) and floating-point numbers.
* **Python Lists:** Understanding how to store and organize data in sequences.
* **Index Errors:** Learning how to avoid `IndexError` and how offset-by-one errors occur.
* **Nested Lists:** Working with lists within lists (matrices).
* **Module Imports:** Incorporating external modules to extend Python's functionality.

### How to Play:
Run `RockPaperScissors.py` in your terminal. Type 0 for Rock, 1 for Paper, or 2 for Scissors. Can you beat the computer's random choice?
