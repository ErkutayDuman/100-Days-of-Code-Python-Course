# Day 5 - PyPassword Generator

This project is part of Dr. Angela Yu's "100 Days of Code: The Complete Python Pro Bootcamp".

### Key Concepts Learned:
* **For Loops:** Iterating over lists to perform actions on each item.
* **Range Function:** Generating sequences of numbers for loops and specific calculations.
* **Accumulator Pattern:** Using variables to keep track of running totals or building strings/lists.
* **Random Shuffling:** Using `random.shuffle()` to increase the security and randomness of the output.
* **Logic with Loops:** Combining `if/else` statements inside loops to solve complex problems.

### How it Works:
The program asks for the desired number of letters, symbols, and numbers, then generates a strong, randomized password based on those inputs.

### How to Run:
Run `PyPasswordGenerator.py` in your terminal and follow the prompts to generate a secure password.
