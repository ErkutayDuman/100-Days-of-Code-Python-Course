# Day 2 - Tip Calculator 

This project is part of Dr. Angela Yu's "100 Days of Code: The Complete Python Pro Bootcamp".

### Key Concepts Learned:
* **Data Types:** Understanding Strings, Integers, Floats, and Booleans.
* **Type Conversion:** Converting between different data types (casting).
* **Mathematical Operations:** Using `+`, `-`, `*`, `/`, `**` and understanding priority.
* **F-Strings:** A powerful way to format strings in Python.

### How it works:
The program calculates the total amount each person should pay, including a tip, by taking the bill amount, the tip percentage, and the number of people as input.
