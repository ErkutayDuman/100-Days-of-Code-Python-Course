# Day 6 - Escaping the Maze 

This project is part of Dr. Angela Yu's "100 Days of Code: The Complete Python Pro Bootcamp". 

### Key Concepts Learned:
* **Defining Functions:** Learning how to create and call custom functions (`def`).
* **While Loops:** Understanding how to execute code as long as a condition remains true.
* **Indentation:** Mastering Python's strict spacing rules within nested functions and loops.
* **Algorithm Design:** Developing a "Follow the Right Wall" logic to solve randomized maze patterns.

### Platform:
This project was developed and tested on [Reeborg's World](https://reeborg.ca/reeborg.html). The code provided in `EscapingTheMaze.py` is designed to run within that environment's API (e.g., `move()`, `turn_left()`, `front_is_clear()`).

### Final Challenge:
The goal was to help Reeborg escape a maze with randomized start positions, using a single, robust algorithm.
