# Day 3 - Treasure Island 

This project is part of Dr. Angela Yu's "100 Days of Code: The Complete Python Pro Bootcamp".

### Key Concepts Learned:
* **Control Flow:** Learning how to use `if`, `elif`, and `else` statements to direct program execution.
* **Comparison Operators:** Using `>`, `<`, `==`, `!=` etc. to evaluate specific conditions.
* **Logical Operators:** Understanding the theory and use of `and`, `or`, and `not` to combine multiple conditions.
* **Nested If/Else:** Structuring decision trees where one choice leads to another set of conditions.
* **String Methods:** Using `.lower()` to ensure user input is handled correctly and consistently.

### Custom Story Logic:
While using the provided ASCII art from the course, I designed my own unique story paths and choice outcomes to practice structuring complex decision-making in Python.

### How to Play:
Run `TreasureIsland.py` in your terminal and see if you can find the treasure without falling into a trap!
