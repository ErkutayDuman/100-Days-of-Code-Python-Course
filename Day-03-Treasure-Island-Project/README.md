# Day 3 - Treasure Island 

This project is part of Dr. Angela Yu's "100 Days of Code: The Complete Python Pro Bootcamp".

### Key Concepts Applied:
* **Conditional Logic:** Built a decision-based game using `if`, `elif`, and `else`.
* **Nested Structures:** Created multiple layers of choices to practice code flow.
* **String Normalization:** Used `.lower()` to ensure user input is handled correctly regardless of case.

### Custom Story Logic:
While using the provided ASCII art from the course, I focused on **designing my own story paths**. I redesigned the choices and outcomes to practice structuring complex decision trees in Python.

### How to Play:
Run `TreasureIsland.py` in your terminal and see if you can find the treasure without falling into a trap!
